# Phase 04 - Home Page

## Purpose
Build the high-fidelity home page using locked design tokens.

## Inputs
- Locked system tokens from Phase 03
- Personas and target audience
- Strategy and positioning
- Current website analysis
- Brand assets

## Process
1. Structure planning
   - Hero section with primary CTA
   - Value proposition sections
   - Social proof/testimonials
   - Service/product highlights
   - Footer with navigation

2. Content creation
   - Write real copy (no lorem ipsum)
   - Pull from strategy pillars
   - Match brand voice
   - Focus on primary persona

3. Visual implementation
   - Apply design tokens consistently
   - Use real imagery if available
   - Create labeled placeholders if not
   - Ensure responsive design

4. Refinement
   - Review against intake decisions
   - Check accessibility
   - Test on multiple viewports
   - Iterate based on feedback

## Outputs
- `output/Homepage.html`
  - Fully styled, responsive page
  - Real copy throughout
  - All CTAs functional
  - Production-ready code

## Exit Criteria
- Page matches all Phase 01 decisions
- Uses only locked tokens from Phase 03
- All copy is real and on-brand
- User approval after revisions
