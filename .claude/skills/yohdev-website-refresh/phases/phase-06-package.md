# Phase 06 - Package

## Purpose
Assemble the final deliverable package for client handoff.

## Inputs
- All outputs from Phases 01-05
- Original Sales Handoff folder
- Client branding

## Process
1. Customize Playbook
   - Apply client brand tokens
   - Insert client-specific content
   - Add project timeline
   - Include decisions and rationale

2. Organize deliverables
   - Copy final Homepage.html
   - Copy final Style Guide.html
   - Copy customized Playbook.html
   - Organize all assets
   - Archive original brief

3. Create documentation
   - Write deployment README
   - List all placeholders/stubs
   - Document next steps
   - Include technical requirements

4. Final quality check
   - Validate all HTML
   - Check all links
   - Verify asset paths
   - Test deployment locally

## Outputs
Final folder structure:
```
{{client-slug}}-website-refresh/
├── Homepage.html
├── Style Guide.html
├── Playbook.html
├── assets/
│   ├── logos/
│   ├── imagery/
│   └── fonts/
├── theme/
│   └── theme.json
├── brief/
│   └── [original handoff files]
├── output/
│   └── [all phase outputs]
├── README.md
└── .nojekyll
```

## Exit Criteria
- All files in correct locations
- README complete with instructions
- Package ready for deployment
- No blockers for client handoff
