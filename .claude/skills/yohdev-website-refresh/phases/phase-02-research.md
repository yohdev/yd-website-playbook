# Phase 02 - Research

## Purpose
Understand the current state and gather visual/content references.

## Inputs
- Intake summary from Phase 01
- All 8 Sales Handoff MD files
- Current website URL
- Competitor/reference sites

## Process
1. Crawl current website
   - Capture screenshots at multiple viewports
   - Extract existing copy and messaging
   - Identify current navigation structure
   - Note what's working well

2. Research competitors and references
   - Screenshot competitor sites
   - Analyze visual patterns
   - Document differentiators

3. Create inventory
   - Visual elements to keep
   - Copy/messaging to preserve
   - Elements to replace or improve
   - Missing elements to add

## Outputs
- `output/02-research.md`
  - Current site analysis
  - Competitor analysis
  - Visual inventory
  - Copy inventory
  - Recommendations

- `output/02-screenshots/`
  - Current site screenshots
  - Competitor screenshots
  - Reference site screenshots

## Exit Criteria
- Complete site audit documented
- All screenshots captured
- Clear recommendations for Phase 03
- User approval to proceed
